LOT 空间实验室

建筑 / 景观 / 城市

 

空间实验室源自于对空间问题以及解决方案的探讨，同时展开对未来城市空间的设想。我们希望通过各种规模和尺度的空间设计和规划方案，来创造各类具有行为效率价值和文明精神价值的人类活动空间。

TOP 产品实验室

WEB / APP / ID

 

产品实验室源自于对互联网技术和工业设计在应用场景的探讨，同时展开对智能化万物联网时代的研究。我们希望通过研发各种创新型软件硬件，来打造可以提高行为效率和生活质量的产品。

DOC 品牌实验室

品牌设计 / 数字营销

 

品牌实验室源自于对市场运营以及品牌价值的探讨，同时展开对各种消费行为和消费心理的分析。我们希望通过符号化的视觉印象和口碑式的营销传播，来培育具有商业价值和文化价值的品牌形象。

SEE 视效实验室

视觉特效 / 影视制作

 

视效实验室源自于对影像技术以及作品艺术的探讨，同时展开对商业化视觉消费的研究。我们希望通过平面和动态的视觉特效和影视后期，来创作具有商业价值和艺术价值的视觉影像作品。
